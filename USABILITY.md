# First-user and accessibility validation

Run this study with at least five people who have not seen Brandrail. Do not coach terminology. Record completion, time, wrong turns, confidence, and the first phrase they use for each concept.

## Tasks

1. Explain what Brandrail does from the landing page in one sentence.
2. Compile a brand and identify what can and cannot be changed.
3. Connect an agent with the minimum permissions and prove the connection works.
4. Create a dry campaign plan and explain why production is paused.
5. Approve the plan, produce an asset, and locate the exact output later.
6. Send work to review, distinguish approval from publishing, and find what needs human action.
7. Schedule approved work, then cancel it safely.
8. Recover from an expired credential, disconnected channel, failed delivery, and stale plan.

Release thresholds: at least 4/5 participants finish tasks 1–6 without intervention; nobody believes plan approval publishes content; nobody exposes a credential; median agent connection is under five minutes; every participant can locate failed work and state the next action.

## Accessibility pass

Test at 390 px and desktop widths, 200% zoom, reduced motion, high contrast, keyboard only, VoiceOver/Safari, and NVDA/Firefox or JAWS/Chrome.

- The first Tab exposes “Skip to main content.”
- Navigation, details menus, permission checkboxes, tabs, dialogs, review actions, and calendar controls are reachable and visibly focused.
- Dialogs announce their title, close with Escape, retain a clear close control, and report failures through an alert region.
- Every input has a programmatic label; every status change is polite unless it blocks progress; every image has useful or empty alternative text.
- Heading order, landmarks, link names, table headers, contrast, target size, zoom/reflow, and horizontal overflow meet WCAG 2.2 AA.

File issues with the route, viewport, assistive technology, exact control name, expected behavior, and severity. A critical publishing, credential, or approval misunderstanding blocks release.
